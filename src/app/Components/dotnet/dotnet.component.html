What is .NET?

.NET Core is a general-purpose development platform maintained by Microsoft and the .NET community on GitHub. It is an open-source, cross-platform framework that supports Windows, macOS, and Linux operating systems. It can be used to build device, cloud, and IoT applications.

Written from scratch to make it modular, lightweight, fast, and cross-platform, .NET Core applications speed up performance, reduce memory footprint, and are easy to maintain. As it includes core features required to run a basic .NET Core app, other features are provided as NuGet packages, which users can add to applications as needed.
